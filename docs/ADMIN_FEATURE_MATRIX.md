# Admin Feature Matrix

## 1. Dashboard Overview
- **KPI Cards:** Total Users, Active Students, Revenue (Monthly/Yearly), Server Status.
- **AI Copilot Widget:** Chat interface to query system stats ("How many students failed math today?").

## 2. User Management
- **List View:** Searchable/Filterable table of all users.
- **Actions:** Create User, Edit Profile, Reset Password, Ban/Suspend, Assign Role.
- **Bulk Import:** Upload CSV to create students/teachers.

## 3. LMS Management
- **Course Manager:** Create/Edit Courses, Drag-and-drop Lesson Builder.
- **Question Bank:** CRUD operations for questions, Bulk Upload, AI Generation tool.
- **Quiz Manager:** Assemble quizzes from the bank, set timers and passing scores.

## 4. Multi-tenancy
- **School Manager:** Create Schools, Assign Supervisors, Set Student Limits.
- **Group Manager:** Organize students into classes.

## 5. Marketing & CMS
- **Banners:** Manage homepage sliders/announcements.
- **Coupons:** Create discount codes.
- **Testimonials:** Approve/Reject user reviews.

## 6. Logs & Security
- **Audit Log:** Who did what and when.
- **Error Logs:** Backend error reporting.
